Mall Customer Analysis
## About
This project performs **Exploratory Data Analysis (EDA)** on mall customer data and groups customers based on their **income and spending behavior**.
## Dataset

* Mall customer dataset
* Features:
  * Age
  * Gender
  * Annual Income
  * Spending Score

## Tools Used
* Python
* Pandas
* Matplotlib
* SQLite
* Scikit-learn

## What I Did
* Cleaned and explored the dataset
* Visualized:
  * Age distribution
  * Income distribution
  * Spending patterns
* Analyzed data using SQL queries
* Applied **KMeans clustering** to segment customers

## Key Insights
* Customers have different spending habits based on income
* Some high-income customers spend less
* Customer segmentation helps identify target groups

## How to Run

```bash
pip install pandas matplotlib scikit-learn
python eda.py
```

## Files
* `eda.py` → Main code
* `Mall_Customers.csv` → Dataset
* `processed_mall.csv` → Output
* `mall.db` → SQL database
